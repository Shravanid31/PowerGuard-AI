# PowerGuard-AI
## Intelligent Power System Fault Detection & Classification using Machine Learning
Machine Learning based Power System Fault Detection and Classification using Flask and IBM Cloud Lite.


## Project Overview

PowerGuard AI is a Machine Learning-based web application developed to detect and classify faults in power distribution systems. The system predicts different fault types using electrical parameters and provides an analytics dashboard for data visualization.

The project is developed using Python, Flask, Random Forest, and IBM Cloud Lite services.

---

## Problem Statement

Power distribution systems are prone to faults such as Transformer Failure, Overheating, and Line Breakage. Manual fault detection is time-consuming and may result in delayed maintenance and power outages.

PowerGuard AI predicts power system faults using Machine Learning to support faster fault identification and improved maintenance planning.
